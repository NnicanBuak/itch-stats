# Itch Stats

UserScript для `itch.io`, который:

- ищет ваши игры в списках `itch.io`
- сохраняет позиции
- показывает статистику
- пассивно подсвечивает найденные игры

## Требования

- браузер **Google Chrome**, **Microsoft Edge**, **Firefox** или другой совместимый
- расширение **Tampermonkey**
- файл скрипта [`ItchStats.js`](./ItchStats.js)

## Установка через Tampermonkey

1. Установите **Tampermonkey**:
   - Chrome / Edge: [Chrome Web Store](https://chromewebstore.google.com/)
   - Firefox: [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/tampermonkey/)
2. Откройте панель **Tampermonkey**.
3. Нажмите **Create a new script**.
4. Удалите шаблонный код из редактора.
5. Откройте файл [`ItchStats.js`](./ItchStats.js), скопируйте всё содержимое и вставьте в редактор Tampermonkey.
6. Нажмите **File -> Save** или `Ctrl+S`.
7. Убедитесь, что скрипт **включён** в списке Tampermonkey.

## Как использовать

1. Откройте `https://itch.io/` и войдите в свой аккаунт.
2. Перейдите на страницы `itch.io`, где скрипт должен работать.
3. Скрипт активируется автоматически, так как он подключается к:
   - `https://itch.io/*`
   - `https://*.itch.io/*`

## Обновление скрипта

Если вы изменили локальный файл `ItchStats.js`, Tampermonkey **не подтянет изменения сам**, если скрипт был вставлен вручную.

Чтобы обновить его:

1. Откройте скрипт в Tampermonkey на редактирование.
2. Замените старый код новым содержимым из [`ItchStats.js`](./ItchStats.js).
3. Сохраните изменения.
4. Обновите страницу `itch.io`.

## Возможные проблемы

**Скрипт не работает**

- проверьте, что он **включён** в Tampermonkey
- проверьте, что страница открыта на домене `itch.io`
- после установки обновите вкладку сайта

**Изменения не появились**

- убедитесь, что в Tampermonkey сохранена **последняя версия** кода
- попробуйте перезагрузить страницу с очисткой кэша

**Tampermonkey не даёт вставить код**

- создайте новый скрипт ещё раз и полностью удалите стартовый шаблон перед вставкой
